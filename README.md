# SimTrack Coach

Starter project scaffold for a Next.js + Supabase sim racing app.

## Included
- app/page.js
- lib/supabase.js
- .env.example
- package.json
- supabase/schema.sql

## Local run
1. npm install
2. create .env.local from .env.example
3. npm run dev

## Better path on older Macs
Upload this repo to GitHub and deploy via Vercel instead of running locally.
